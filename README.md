# Twitter Sentiment Analysis using BiLSTM

This project performs sentiment analysis on tweets using a BiLSTM deep learning model.

## Features
- Classifies tweets as Negative, Neutral, or Positive
- NLP preprocessing and tokenization
- Trained using TensorFlow and Keras
- Deployed using Streamlit

## Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Streamlit

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py

## Sample Input
"I love this new update, it's amazing!"

## Output
Sentiment: Positive
